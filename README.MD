# Todo-Scheduler
> 우선순위 기반 자동 일정 추천 플래너    
## 📌 주요 기능
- 회원가입 / 로그인 (JWT)
- 일정 CRUD
- 자동 일정 추천 (우선순위 + 여유 시간 기반)
- 타임라인 / 캘린더 뷰
- 태그 / 우선순위 필터

## 🛠 기술 스택
- Frontend: React + TypeScript
- Backend: Node.js (Express)
- DB: MySQL (Docker)
- Auth: JWT
